## ğŸ¤– ROBOTS.TXT FILE

### **Purpose:**
The `robots.txt` file tells search engine crawlers which pages they can and cannot access on your site.

### **Content:**

```txt
User-agent: *
Allow: /

Disallow: /*.md$

Sitemap: https://www.freebodyperformance.com/sitemap.xml
```

### **What It Does:**

**Allow All Crawlers:**
```
User-agent: *
Allow: /
```
- Allows all search engines (Google, Bing, etc.)
- Crawl all pages on the site

**Disallow Documentation:**
```
Disallow: /*.md$
```
- Blocks crawling of .md files (documentation)
- Keeps internal docs private

**Sitemap Location:**
```
Sitemap: https://www.freebodyperformance.com/sitemap.xml
```
- Points crawlers to your sitemap
- Helps with efficient crawling
